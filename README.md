CSS-Label-Generator
===================

CSS Label Generator to rapidly create a label to use in your site. Try it for yourself at http://akerman-portfolio.com/labelgen/


Current Features
----------------

* Label Height
* Label Width
* Position Left/Right
* Fill Colour
* Border On/Off
* Border Width
* Border Colour
* Font Size
* Font Colour
* Message

Feel free to fork project and make any improves.
